# CCTV Site Planner Pro

A lightweight, browser-based tool for planning CCTV camera placement on real site maps.

## What it does
- Import a site map or floor plan
- Drop cameras visually on the map
- Set type, angle, field of view, and range
- Calibrate real-world scale in metres
- Save, load, and export camera configurations
- Undo and redo changes safely

## Why it exists
This tool was built to make CCTV planning faster, clearer, and less abstract.
If you can see the site, you can design the coverage.

## How to use
Hosted via GitHub Pages.  
Open in a desktop browser for best results.

## Tech
- Plain HTML, CSS, and JavaScript
- Canvas-based rendering
- Local storage and JSON export

## Status
Actively usable.
Open to improvement.
Built for real-world security planning, not demos.
